dotfiles
========

Inspired by [Harry's dotfiles](https://github.com/hmarr/dotfiles) and [Phil's dotfiles](https://github.com/Harrison/dotfiles).

## Installation

### OS X Config

To install

```bash
$ ./osx
```

### Terminal config

To install (make sure you have Homebrew and go)
```bash
$ git submodule update --init
$ go run update.go
```
